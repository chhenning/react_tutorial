# Shortcuts

Ctrl W -- Close Current Document

Ctrl Shift V -- Preview markdown

Ctrl F4 -- Close current tab

# Zen Coding

Shortcut to create a complex html structure

```
bla(){
  table.table>thead>tr>td*4 THEN TAB
}
```

will expand into

```
bla(){
  <table className="table">
    <thead>
      <tr>
        <td></td>
        <td></td>
        <td></td>
        <td></td>
      </tr>
    </thead>
  </table>
}

```
